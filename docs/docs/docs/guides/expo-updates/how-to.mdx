# How To Use Expo Updates

:::tip Expo Updates are supported out of the box with Expo Config Plugin by `@callstack/react-native-brownfield`.
:::

In only two steps, you will get the Expo Updates up and running in isolated brownfield projects:

#### Step 1

- Complete the `@callstack/react-native-brownfield` Expo Config Plugin [setup](/docs/getting-started/expo)

#### Step 2

- Complete the [Expo Updates setup](https://docs.expo.dev/eas-update/getting-started/)

<hr/>

To see a demo integration, please visit:

- [ExpoApp56](https://github.com/callstack/react-native-brownfield/tree/main/apps/ExpoApp56)

<hr/>

##### SDK 55

:::tip iOS
Supported out of the box from version `expo-updates@55.0.21` and onwards.
:::

:::warning Android
The support is in development by the Expo Team. In the meantime, checkout [this](/docs/guides/expo-updates/expo-55) manual guide.
:::
